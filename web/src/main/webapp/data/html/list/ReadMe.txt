
增加系统模型信息列表样式说明，每个系统模型的列表共四个文件，分别如下：(“信息列表名称”在后台系统模型里设置)
--------------------------------------------------
信息列表名称.php           后台按栏目管理信息列表样式
all信息列表名称.php        后台按表管理信息列表样式
doc信息列表名称.php        后台管理归档信息列表样式
q信息列表名称.php          前台会员管理信息列表样式
--------------------------------------------------
(附加说明：上面四个列表文件不是必须增加，不加文件则系统会使用默认列表。)



例如默认的列表：listinfo
--------------------------------------------------
listinfo.php
alllistinfo.php
doclistinfo.php
qlistinfo.php
--------------------------------------------------

例如信息列表名称为：news
--------------------------------------------------
news.php
allnews.php
docnews.php
qnews.php
--------------------------------------------------
